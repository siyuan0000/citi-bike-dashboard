# Citi Bike Dashboard

A Flask-based web application for monitoring Citi Bike station status in real-time and visualizing historical ride analytics.

![Architecture Diagram](files/citibike-dashboard-architecture.png)

## Features
- **Real-Time Map**: Live monitoring of Citi Bike station availability via GBFS data integration.
- **Historical Analytics**: Dashboard displaying pre-computed ride statistics generated by PySpark.

## Prerequisites
- Python 3.9+

## Quick Start (Local)

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the local environment:**
   ```bash
   bash startLocal.bash
   ```
   *(Alternatively, run `python app.py`)*

3. **View the application:**
   Open [http://localhost:5000](http://localhost:5000) in your web browser.

## Cloud Deployment (AWS)
To run the server with AWS RDS (MySQL) and S3 integration, define your environment variables (or let EB handle them) and use the cloud startup script:
```bash
bash startCloud.bash
```
